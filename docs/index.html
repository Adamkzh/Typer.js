Getting Started

Here's how to hit the ground running: it's very simple, as all of Typer.js is packaged in one short javascript file.

Adding a Cursor

To add a cursor, add a tag with the class "cursor" after your typer class. The cursor is kept as a separate tag so that your typer and cursor can be configured to your liking in a span, p, or even div.

class typer

Options
data-words
 A comma-delimited list of words to be typed.
Default: 'Sample typing'
data-delim Specifies the delimter used in data-words
Default: ,
data-delay The time delay between typed characters.
Default: 200
id (potential dependency) A unique identifier that can be used to set the owner property of other objects. This must be set if a cursor, typer-start, or typer-stop is used with this typer.
data-deleteDelay The time delay before a completed word is de-typed.
Default: 800
data-colors A comma-delimited list of colors to cycle through. One color per word.
Default: black

class cursor

Options
data-owner (required) Specifies the id of the typer DOM object associated with this cursor.
data-cursorDisplay The cursor representation.
Default: _

class typer-start

Options
data-owner (required) Specifies the id of the typer DOM object associated with this button.

class typer-stop

Options
data-owner (required) Specifies the id of the typer DOM object associated with this button.
